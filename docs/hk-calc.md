# 港股计算器文档

## 计算交易手续费
```
输入：总金额,免佣,窝轮|正股
输出：手续费

计算规则：
  a. 计算交易总额
  b. 根据a的结果-->获取券商交易规则
  	b.1: 没有合适的规则，return 0;
  	b.2: 进入下一步.
  c. 判断佣金方案是否为免佣金
  d. 计算佣金
  e. 计算平台使用费
  f. 计算交易系统使用费
  g. 计算交收费
  h. 计算交易费
  i. 计算交易征费
  j. 计算印花税
  
  k. 计算总费用
    k.1: 股票类型 === 窝轮(不需要交 e项印花税)
      return d+f+g+h+i+j;
    k.2:
     	return d+e+f+g+h+i+j;
```
